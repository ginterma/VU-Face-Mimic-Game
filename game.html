EMOTION VALUE

Face mimic game!

The purpose of this game is to mimic displayed emotion on the right and aim to achieve as high as possible High Score.
With each consecutive level, difficulty of game increases.



EMOTION VALUE

How to play:

* Press Start Game button.
* During given time limit take photo of your emotion by clicking Take Photo button on camera view.
* If both emotions are similar enough then player proceeds to the next level by pressing Next Level button.
* If emotions are not similar enough then player can start again from level 1 by pressing Start Game button.

Good luck!
